# Deal Surgeon

Deal Surgeon is an experimental lab focused on building AI-powered revenue systems and automation tools for modern GTM teams.

The goal is to explore how workflow automation, AI agents, and data pipelines can improve sales execution and pipeline visibility.

## Projects

### Inbound Lead Qualifier
Automated inbound lead qualification and routing system built with n8n.

Repository:
https://github.com/caleb-matkovich/inbound-lead-qualifier

### Deal Risk Monitor
